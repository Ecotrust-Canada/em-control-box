
ADDING CARD PARAM VIA MODPROBLE
	modprobe -r bt878 (because the bttv module was used by that module)
	modprobe -r bttv
	modprobe bttv card=77
	modprobe bt878

	To make this parameter adding persistent:
	Add "options bttv card=77" (77 was the value that worked for me) in /etc/modprobe.d/options.
	
	options i2c-algo-bit bit_test=1
